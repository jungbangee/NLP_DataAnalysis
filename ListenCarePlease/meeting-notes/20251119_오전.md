### 전사 요약 기능

- 기존 템플릿 기반 요약 1단계 구현 계획 + 조직에서 사용 중인 템플릿에 맞춰 회의록 요약 기능 우선 개발 + 템플릿이 없는 경우나 새로운 템플릿 적용은 2단계 이후 고려
- 템플릿 추천 및 고도화 방향 + 회의 내용 분석 후 적합한 템플릿 추천 알고리즘은 3단계로 계획 + 1단계 완료 후 2단계, 3단계 점진적 개발로 난이도 분산
- 난이도와 우선순위 조정 + 1단계만 완료해도 기본 기능 확보 가능 + 2단계는 텍스트 정리 및 변환 작업으로 2~3주 소요 예상 + 팀 내 역할 분담과 일정 조율 필요

- 평가 가이드라인 부재 및 조사 진행 + 현재 구체적 가이드라인은 없으며 민서님이 관련 자료 조사 중

- 검색과 주석 기능 개념 구분 + 검색은 음성파일 내 단어 검색 기능으로 이해 + 주석은 문맥 파악을 통한 키워드, 전문 용어, 논란 용어에 대한 설명 부착
